# Sample project for playing with Django

## List of commands for initial setting

- `python --version`
- `pip install pipenv`
- `cd C:\dev\`
- `mkdir django`
- `ls`
- `cd .\django\`
- `mkdir storefront`
- `cd .\storefront\`
- `pipenv install django`
- `ls`
- `code .`
- `pipenv shell`

## Virtual env commands

- `cls`
- `django-admin`
- `django-admin startproject storefront`
- `django-admin startproject storefront .`
- `django-admin runserver`
- `python .\manage.py`
- `python .\manage.py runserver`
